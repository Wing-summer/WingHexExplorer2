## 最新发布日志

&emsp;&emsp;该仓库仅提供基于 QT6 的十六进制编辑器安装包，对于 Windows 采用 InnoSetup 安装包。对于 Linux 采用 run 自解压安装文件进行安装，请在终端使用命令行使用 root 权限进行，若要卸载请到`/opt/WingHexExlorer2`下进入终端使用 root 权限执行`uninstall.sh`完成卸载。 **在 Linux 上，如果你有对应的编译环境，我不推荐使用 run 安装文件方式。** 因为为了保证安装包的通用性，我在里面部署了大量的库，包括对应的 glic 和 ld-linux和对可执行文件做了`patch`，让文件总体积膨胀，这是一个很不划算的事情。建议使用`mkinstaller`下的`pyscript`进行安装。

&emsp;&emsp;**本软件不会以任何理由主动向你收取费用，你通过下载渠道给的费用一分钱也不会到我这里，请注意。**

&emsp;&emsp;如有 Bug ，请到`Github`或者`Gitee`提交`issue`。如果有功能建议，也可以通过`issue`，不过我更建议 **到 Github 的 discussion 区提交** 。

&emsp;&emsp;该工具的定位是对二进制的编辑和分析。**欢迎热心的大佬一起共建基础开源工具**。

&emsp;&emsp;目前只有我一个人开发，我不可能能够把每个功能做好测试，尤其是做 **高风险** 的操作，点名在 root 权限下操作文件。

&emsp;&emsp;我可能不会对每个 Bug 能够积极响应，不过对大家的建议和需求我会尽可能地尽早回复和评估处理。

## 更新内容

&emsp;&emsp;该次更新更新有功能优化和缺陷修复，以下是相对于上一个版本的增强和修复日志：

* 【严重】修复单个模式下脚本竞争运行的崩溃问题
* 【严重】修复插件卸载中调用服务崩溃的问题
* 【严重】修复 WingAngelAPI 在未初始化完毕就请求执行脚本的崩溃问题
* 【中等】修复插件脚本函数使用安全方式注册 dictionary 作为返回值失败的问题
* 【中等】修复代码预处理器无法正常工作和脚本全局变量无法初始化的问题
* 【中等】修复插件组件 API 调用被拒绝的问题
* 【轻微】修复脚本编辑窗口退出时无法销毁的问题
* 【轻微】修复颜色选择框无法修改标题栏文本以及改进修复颜色滑动条
* 【轻微】更好的处理控制台命令换行问题
* 【轻微】修复从历史记录打开文件的设置保存未和操作同步的问题
* 【轻微】修复在脚本挂载调试器模式下无法打印异常信息的问题
* 【轻微】修复脚本编辑含有下划线的标识符无法代码填充的问题
* 【轻微】修复十六进制编辑菜单中的上下文过早的执行结束相关代码
* 【调整】现在 Toast 消息可以随窗口状态更新位置，鼠标悬浮可以暂停消失计时器
* 【调整】轻微优化 API 调用开销
* 【调整】重构代码预处理器和预处理计算器，进一步严格和限缩保证健壮性和报错用户友好
* 【调整】IWingDevice 接口现在可以控制默认的文档打开状态
* 【调整】补充缺失的标注相关 API 接口，使插件或者脚本更加充分掌控
* 【调整】现在在有依赖的插件可视化以及启用禁用更加用户体验友好
* 【调整】使用增强的 angel-lsp 的语法高亮（控制台只使用基本高亮）
* 【调整】现在 angel-lsp 的预处理器有了更好的支持
* 【调整】补充了遗漏的脚本语言的`>>>`操作符语法解析支持
* 【调整】十六进制编辑文件窗口选项卡会显示和历史记录一样的详情信息
* 【调整】默认情况下 WingCStruct 插件是启用的
* 【调整】废弃脚本编辑器的“长线边缘”属性设置支持
* 【调整】Snippet 预处理器现在会自动填充默认值，而不是忽略
* 【调整】优化插件系统的事件派发和关键事件看门狗保护
* 【调整】优化了脚本编辑器的输出 alt 定位可视化操作
